双指针法解决“盛最多水的容器”问题的贪心合法性证明

1. 问题重述与符号定义

设数组 height 长度为 n，索引从 0 到 n-1。
定义容器由两条线 i 和 j 构成（i < j），其水量为：

[
S(i, j) = \min(\text{height}[i], \text{height}[j]) \times (j - i)
]

双指针法初始：left = 0，right = n-1。

2. 核心引理

引理 1：
在任意状态 (left = i, right = j)，假设 height[i] ≤ height[j]（即左边是短板），那么对于所有以 i 为左边界的容器 (i, k)（其中 i < k < j），它们的水量都不可能大于当前容器 (i, j) 的水量。

引理 2：
对称地，如果 height[j] < height[i]（右边是短板），那么对于所有以 j 为右边界的容器 (k, j)（其中 i < k < j），它们的水量都不可能大于当前容器 (i, j) 的水量。

3. 引理的证明

3.1 引理 1 的证明

已知：
[
S(i, j) = \min(\text{height}[i], \text{height}[j]) \times (j - i)
]
因为 height[i] ≤ height[j]，所以：
[
\min(\text{height}[i], \text{height}[j]) = \text{height}[i]
]
因此：
[
S(i, j) = \text{height}[i] \times (j - i)
]

现在看 S(i, k)，其中 i < k < j：
[
S(i, k) = \min(\text{height}[i], \text{height}[k]) \times (k - i)
]

分两种情况讨论：

情况 1：如果 height[k] ≥ height[i]，则：
[
\min(\text{height}[i], \text{height}[k]) = \text{height}[i]
]
所以：
[
S(i, k) = \text{height}[i] \times (k - i)
]
因为 k < j，所以 (k - i) < (j - i)，因此：
[
S(i, k) < \text{height}[i] \times (j - i) = S(i, j)
]

情况 2：如果 height[k] < height[i]，则：
[
\min(\text{height}[i], \text{height}[k]) = \text{height}[k]
]
所以：
[
S(i, k) = \text{height}[k] \times (k - i)
]
因为 height[k] < height[i] 且 (k - i) < (j - i)，所以：
[
S(i, k) < \text{height}[i] \times (j - i) = S(i, j)
]

结论：无论哪种情况，都有 S(i, k) < S(i, j)。
即：当前短板 i 固定时，当前容器 (i, j) 已经是以 i 为左边界的最大可能水量。

3.2 引理 2 的证明

证明过程与引理 1 完全对称，只需交换 i 和 j 的角色即可。

4. 贪心选择的合法性

基于以上引理，我们可以推出：

  • 当 height[i] ≤ height[j] 时，以 i 为左边界的最大可能容器已经被考虑过（即当前容器 (i, j)）。
    因此，继续保留 i 作为左边界去尝试其他 (i, k) 是无意义的，因为这些都不可能比当前大。
    所以可以安全地放弃 i，将左指针向右移动一步（i++）。

  • 当 height[j] < height[i] 时，同理，可以安全地放弃 j，将右指针向左移动一步（j--）。

5. 归纳法证明算法能找到全局最优解

5.1 基础情况

初始状态 (0, n-1) 是当前考虑的最宽容器。此时我们还没有排除任何可能的解。

5.2 归纳假设

假设算法当前状态是 (i, j)，并且我们已经检查了所有可能的容器中宽度大于 (j-i) 的那些组合。

5.3 归纳步骤

根据引理，至少有一个边界（短板的那一边）不会再产生更优解。具体来说：

  • 如果 height[i] ≤ height[j]，则所有以 i 为左边界的容器中，(i, j) 已经是最大的。因此，我们可以安全地将左指针右移一位，不会错过全局最优解。
  • 如果 height[j] < height[i]，则所有以 j 为右边界的容器中，(i, j) 已经是最大的。因此，我们可以安全地将右指针左移一位，不会错过全局最优解。

移动指针后，新的状态 (i', j') 继续满足归纳假设：所有宽度大于 (j' - i') 的容器都已经被考虑过。

5.4 终止条件

当 i ≥ j 时，所有可能的容器组合都已通过上述逻辑被“覆盖”过（每个左边界或右边界被放弃时，其最优候选已被考虑）。因此，过程中记录的最大值就是全局最大值。

6. 直观理解

为什么这个贪心是对的？
因为容器的高度由短板决定。如果你固定短板，无论另一条线怎么移动（只要还在内部），宽度减少，高度不变或降低，水量必然减少。
所以只有移动短板，才有可能遇到更高的边，从而获得更大的水量。

7. 总结

双指针法的合法性基于这样一个关键观察：

对于当前区间 [i, j]，如果 height[i] < height[j]，那么 (i, j) 已经是所有以 i 为左边界的容器中最大的，因此可以安全地丢弃左边界 i，继续搜索。

这个论证确保了算法的贪心选择不会错过最优解，并且时间复杂度为 O(n)。
